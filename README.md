# appropriate/tcpdump
Alpine-based image with just tcpdump

## Usage

```console
$ docker run --rm --net=host appropriate/tcpdump
```

## Tags

* `appropriate/tcpdump:latest`: based on `alpine:latest`
* `appropriate/tcpdump:edge`: based on `alpine:edge`
